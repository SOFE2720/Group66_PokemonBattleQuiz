In this folder contains all of our programs Component-Level Design features,

The traditional component level design feature which outlines the 
interactions between all the components of our program.

The State diagram feature outlines the detailed interactions between each component
and the interfaces.

The High level sequence diagram outlines the interactions of the user and the program
to provide a broad understanding of the interactions between them

The detailed sequence diagram outlines the interactions of the user and the program with
more detail and provides a excellent understanding of how they interact.
